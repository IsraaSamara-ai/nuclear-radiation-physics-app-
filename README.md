Nuclear Radiation Interactive / الإشعاع النووي التفاعلي
Interactive Educational Application for Nuclear Radiation Physics
تطبيق تعليمي تفاعلي لفيزياء الإشعاع النووي
A comprehensive, interactive digital application designed to teach nuclear radiation concepts to high school students. Features 3D animations, interactive simulations, sound effects, and real-life analogies to simplify complex physics concepts.

تطبيق رقمي تفاعلي شامل مصمم لتعليم مفاهيم الإشعاع النووي لطلاب المرحلة الثانوية. يتميز برسوم متحركة ثلاثية الأبعاد، محاكاة تفاعلية، مؤثرات صوتية، وأمثلة من الحياة اليومية لتبسيط المفاهيم الفيزيائية المعقدة.

Features / المميزات
3D Penetration Comparison: Interactive Three.js visualization comparing alpha, beta, and gamma radiation penetration
Nucleus Decay Animation: 3D visualization of nuclear decay processes
Coin Toss Simulation: Interactive modeling of radioactive decay
Interactive Decay Curves: Plotly-based charts with adjustable parameters
Decay Series Visualization: N-Z diagram of uranium decay series
Sound Effects: Audio feedback for interactive elements
Arabic/English Bilingual: Full support for Arabic text with English symbols
